# 理论上界推导

这个上界只使用总绩点资源，不考虑课程池结构和整数约束，因此只能说明“不可能超过多少”，不能说明一定可以达到。

## 单门课平均绩点

固定成绩分布如下：

| 成绩 | GPA | 每 100 人数量 |
| --- | ---: | ---: |
| C- | 1.7 | 4 |
| C | 2.0 | 5 |
| C+ | 2.3 | 10 |
| B- | 2.7 | 16 |
| B | 3.0 | 20 |
| B+ | 3.3 | 19 |
| A- | 3.7 | 15 |
| A | 4.0 | 8 |
| A+ | 4.3 | 3 |

平均绩点为：

```text
(1.7*4 + 2.0*5 + 2.3*10 + 2.7*16 + 3.0*20 + 3.3*19 + 3.7*15 + 4.0*8 + 4.3*3) / 100
= 3.061
```

## 上界公式

设目标阈值为 `T`，达标学生占比为 `p`。

为了在只考虑总绩点资源时最大化达标比例，可以做最乐观的假设：达标学生刚好拿到平均 `T`，未达标学生全部拿到最低绩点 `1.7`。因此总体平均绩点必须满足：

```text
3.061 >= p*T + (1-p)*1.7
```

整理得到：

```text
p <= (3.061 - 1.7) / (T - 1.7)
```

所以理论上界为：

```text
ceiling(T) = min(1, (3.061 - 1.7) / (T - 1.7))
```

## 阈值 3.5 的结果

当 `T = 3.5`：

```text
ceiling = (3.061 - 1.7) / (3.5 - 1.7)
        = 1.361 / 1.8
        = 0.7561
```

因此，从总绩点资源角度看，平均 GPA 达到 `3.5` 的比例不可能超过：

```text
75.61%
```

如果粗略换算到 300 名学生，对应约：

```text
226 人
```

这是非常乐观的上界。实际问题还存在课程池共享结构、每门课整数成绩人数、不同专业课程数不同等约束，所以真正可达到的人数可能更低。
