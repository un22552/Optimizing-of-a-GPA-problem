# 实际问题定义

## 目标

共有 300 名学生，分属三个专业。每门课的成绩分布固定，学生根据专业不同修读公共课、共享课和专业课。问题是：在不改变每门课成绩分布的前提下，怎样分配成绩，才能让尽量多学生的平均 GPA 达到阈值 `T`。

主要实验阈值为：

```text
T = 3.5
```

## 成绩与绩点

成绩和绩点对应关系如下：

| 成绩 | GPA |
| --- | ---: |
| C- | 1.7 |
| C | 2.0 |
| C+ | 2.3 |
| B- | 2.7 |
| B | 3.0 |
| B+ | 3.3 |
| A- | 3.7 |
| A | 4.0 |
| A+ | 4.3 |

每 100 人课程的固定成绩人数为：

| 成绩 | 人数 |
| --- | ---: |
| C- | 4 |
| C | 5 |
| C+ | 10 |
| B- | 16 |
| B | 20 |
| B+ | 19 |
| A- | 15 |
| A | 8 |
| A+ | 3 |

对应的单门课平均绩点为：

```text
3.061
```

150 人、250 人、300 人课程按同一比例缩放成绩人数。

## 学生分组

| 分组 | 学生数 | 每人课程数 |
| --- | ---: | ---: |
| G1 | 150 | 24 |
| G2 | 100 | 24 |
| G3 | 50 | 28 |

## 课程池

| 课程池 | 适用学生 | 课程数 | 每门人数 | 总槽位 |
| --- | --- | ---: | ---: | ---: |
| 公共课 | G1, G2, G3 | 10 | 300 | 3000 |
| G1-G2 共修课 | G1, G2 | 4 | 250 | 1000 |
| G2-G3 共修课 | G2, G3 | 3 | 150 | 450 |
| G1 专业课 | G1 | 10 | 150 | 1500 |
| G2 专业课 | G2 | 7 | 100 | 700 |
| G3 专业课 | G3 | 15 | 50 | 750 |

总课程槽位：

```text
7400
```

## 各组修课结构

G1 学生：

```text
10 门公共课 + 4 门 G1-G2 共修课 + 10 门 G1 专业课 = 24 门
```

G2 学生：

```text
10 门公共课 + 4 门 G1-G2 共修课 + 3 门 G2-G3 共修课 + 7 门 G2 专业课 = 24 门
```

G3 学生：

```text
10 门公共课 + 3 门 G2-G3 共修课 + 15 门 G3 专业课 = 28 门
```

## 优化形式

给定阈值 `T`，最大化：

```text
平均 GPA >= T 的学生人数
```

约束为：

- 每个课程池必须恰好使用规定数量的各等级成绩。
- 每个学生必须从自己可修的课程池中获得对应数量的成绩。
- 共享课程池的成绩可以分给参与该共享池的任一专业学生。
- 只有当学生总绩点不低于 `T * 课程数` 时，该学生才计为达标。

## 难点

这是一个混合整数线性规划问题。难点来自两个方面：一是每名学生跨多个课程池会形成大量可能的成绩组合；二是公共课和共修课会把三个专业的分配耦合在一起。

代码中把部分大课程池拆成较小子池，以降低组合数量。即便如此，全局精确 MILP 对 CBC 仍然偏重，所以实践中更适合先用启发式搜索共享池分配，再对候选方案做按专业拆分的小 MILP 精修。
